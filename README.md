# Personal Website


## Templates

### React Template
```
git clone https://github.com/yeb9925/react-spa-template.git
cd /react-spa-template
yarn install
yarn start
```

### Vue Template
```
git clone https://github.com/yeb9925/vue-spa-template.git
cd /vue-spa-template
yarn install
yarn start
```

On the browser go to localhost 3000
